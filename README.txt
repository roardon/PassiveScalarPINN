Folder description:
This folder contains all Jupyter Notebooks needed to operate the PINN in various
configurations.
Please read the PINN guide PDF for a more detailed breakdown of the physics and
machine learning fundamentals.
The notebooks within this folder save models to the subfolder named 'models'.
The python package installs required for the PINN can be found within 
'requirements.txt'.
